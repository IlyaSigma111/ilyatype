# 🦝 Ильтест — тест скорости печати от Ильябуги Полимуся

Шуточный тест скорости печати на русском языке. Сайт для GitHub Pages.

## Возможности

- Таймер на 15/30/60/120 секунд
- Подсчёт скорости (знаков в минуту)
- Точность и количество ошибок
- Тексты разной сложности и тематик
- Удобный интерфейс в стиле brainapps.ru

## Запуск

Просто открой `index.html` в браузере.

## Деплой на GitHub Pages

1. Создай репозиторий
2. Залей все файлы
3. В настройках репозитория включи GitHub Pages (ветка main, папка /root)
